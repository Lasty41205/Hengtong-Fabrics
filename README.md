# AI销货单助手

一个面向门店日常开单使用的轻量工具。

它的目标很简单：让店员更快录入订单、生成销货单、查看历史记录、管理客户欠账，减少手写和反复整理的时间。

## 这个产品能做什么
- 快速录入订单内容
- 生成清晰的销货单预览
- 保存并查看历史开单记录
- 管理客户资料和价格信息
- 记录客户欠账、还款和账单变化
- 支持多位店员共用同一套线上数据

## 适合谁用
- 小型门店
- 需要频繁开单的销售场景
- 想把纸质记录逐步改成线上管理的团队

## 日常使用流程
1. 店员登录系统
2. 录入客户和商品信息
3. 生成并确认销货单
4. 需要时回看历史记录
5. 在账单页继续跟进客户欠款和回款

## 当前版本特点
- 页面以移动端使用体验为优先
- 操作尽量简化，适合快速上手
- 历史记录、价格表、账单信息可多人共享
- 适合作为门店实际使用版本，也适合作为作品集展示项目
